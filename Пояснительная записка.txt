Для использования функции просмотра заданий необходимо нажать кнопку "расскажи о заданих" тогда бот выдаст информацию о заданиях, начиная с первого, далее пользователь сам листает с помощью стрелочек
Для использования функции совета нужно нажать кнопку "дай совет" и он выдаст случайный совет из бд по теме огэ по информатики 
Для использования функции решения задач нужно нажать кнопку "случайное задание" и бот выдаст условие задачи по информатики, далее вам необходимо написать ответ или нажать стоп и тогда бот закочнит и выдаст ответ. При неправильном ответе, бот предложить попробовать ещё раз, так пока не будет введён правильный ответ 
Для использования обратной связи нужно нажать кнопку "вопрос модератору" и написать сам вопрос, его получит пользователь бота со статусом модератора, модератору необходимо ответить на вопрос и тогда этот ответ получит пользователь
